# Product Dashboard

## Overview

The Product Dashboard is a web application that displays product information using data retrieved from an external API. The project focuses on presenting business data in a clean, organized, and user-friendly interface.

## Features

* Display product information
* Retrieve data from an external API
* Dynamic content rendering
* Responsive layout
* Simple and organized interface

## Technologies Used

* HTML
* CSS
* JavaScript
* REST API

## How to Run

1. Clone this repository.
2. Open the project in Visual Studio Code.
3. Open the `index.html` file using Live Server.

## What I Learned

Through this project, I gained experience working with APIs, fetching and displaying data, manipulating the DOM, and creating a dashboard that presents information clearly for users.
